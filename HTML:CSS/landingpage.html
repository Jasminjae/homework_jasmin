Latitude Analysis

Purpose of Project: To analyze how weather changes as you get closer to the equator. In order to accomplish this task, we pulled data from the OpenWeatherMap API to assemble a data on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. The four factors that were looked at includes: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, along with explanations and descriptions of any trends and correlations witnessed.
